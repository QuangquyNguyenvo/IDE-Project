## ğŸ› ï¸ Development

### Prerequisites
- Node.js v18+
- npm or yarn

### Setup

```bash
# Clone the repository
git clone https://github.com/QuangquyNguyenvo/IDE-Project.git
cd IDE-Project

# Install dependencies
npm install

# Run in development mode
npm start

# Build for Windows
npm run build:win
```
